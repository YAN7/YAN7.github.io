flex布局学习

写rn要用到flex布局，所以重新温习一下flex布局

写在前面，这篇文章高度借鉴了阮一峰老师的Flex 布局教程：语法篇,这篇文章本意是温习，所以把flex的api照着阮一峰老师的这篇教程过了一遍。

1. 容器的属性

以下6个属性设置在容器上

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

1.1 flex-direction

flex-direction决定主轴的方向，（即项目的排列方向）
它的可能值有四个：

  • row(默认值): 主轴为水平方向，起点在左端；
  • row-reverse: 主轴为水平方向，起点在右端；
  • column: 主轴为垂直方向，起点在上沿；
  • column: 主轴为垂直方向，起点在下沿。

1.2 flex-wrap

默认情况下，项目都排在一条线上（又称“轴线”），flex-wrap定义如果在一条轴线上拍不下，该如何换行。
它的可能值有三个：

  • nowrap(默认值): 不换行；
  • wrap：换行，第一行在上；
  • wrap-reverse：换行，第一行在下。

1.3 flex-flow

flex-flow是flex-direction和flex-wrap的简写（合写），默认值是row nowrap.

1.4 justify-content

justify-content定义了项目在主轴上的对齐方式。
它的可能值有五个，具体对齐方式与轴的方向有关。以下假设主轴方向为从左到右

  • flex-start(默认值): 左对齐；
  • flex-end: 右对齐；
  • center： 居中对齐；
  • space-between: 两端对齐，项目之间的间隔都相等；
  • space-around: 每个项目两侧的间隔相等，所以项目之间的间隔比项目与边框的间隔大一倍。

1.5 align-items

align-items定义了项目在交叉轴上的对齐方式。
它的可能值有五个，同样与轴的方向有关，以下假设交叉轴方向为从上沿到下沿

  • flex-start(默认值): 交叉轴的起点对齐；
  • flex-end(): 交叉轴的终点对齐；
  • center: 交叉轴的中点对齐；
  • baseline: 项目的第一行文字的基线对齐；
  • stretch(默认值): 如果项目未设置高度或者设为auto，将占满整个容器的高度。

1.6 align-content

align-content定义了多根轴线的对齐方式，如果项目只有一根轴线，该属性则不起作用。
它的可能值有六个：

  • flex-start: 与交叉轴的起点对齐；
  • flex-end: 与交叉轴的终点对齐；
  • center: 与交叉轴的中点对齐；
  • space-between: 与交叉轴两端对齐，轴线之间的间隔平均分布；
  • space-around: 每根轴线两侧的间隔相等，所以轴线之间的间隔比轴线与边框的距离大一倍。

2. 项目的属性

以下六个属性可以设置在项目上：

  • order;
  • flex-grow;
  • flex-shrink;
  • flex-basis;
  • flex;
  • align-self.

2.1 order

order定义项目的排列顺序，它的属性值为一个数字，数值越小，排列越靠前，默认为0。

2.2 flex-grow

flex-grow定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。
如果所有项目的flex-frow属性值都为1，它们将平分剩余空间（如果有剩余空间的话），如果一个项目的flex-grow的属性值为2，其它项目的该属性值为1的话，则前者占据的剩余空间将比其它项多一倍。

2.3 flex-shrink

flex-shrink定义了项目的缩小比例，它的属性值同为一个数字，默认为1，则如果空间不足，该项目都缩小。
如果所有项目的flex-shrink的属性值都是1，则空间不足时都将等比例缩小，如果某项目的该属性值为0，其它项目为1，则空间不足时前者不缩小。

2.4 flex-basis

flex-basis定义了在分配多余空间之前，项目占据的主轴空间。浏览器根据这个属性，计算主轴是否有多余空间，它的默认值为auto，将项目的本来大小。
它可以设为和width或height一样的值（如250px），则项目将占据固定空间。

2.5 flex

flex是flex-grow,flex-shrink,flex-basis的简写， 它的默认值是0 1 auto,后两个属性可选。
该属性有两个快捷值：

  • auto (1 1 auto): 自动占据剩余空间，空间不足时等比例缩小（弹性盒子）
  • none (0 0 auto): 不自动占据剩余空间，空间不足时不缩小（固态盒子）

2.6 align-self

`align-self属性允许该项目盒子可以单独设置交叉轴方向的对齐方式，可覆盖align-items属性，默认值为auto，表示继承父盒子的align-items属性，如果没有父元素，则等同于stretch。
除了auto属性值，其它的五个属性值和align-items属性完全一致。
